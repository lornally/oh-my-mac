### 比较简单了, 因为有大神做了各种脚本实现

```sh
# oh-my-zsh
sh -c "$(curl -fsSL https://gitee.com/pocmon/ohmyzsh/raw/master/tools/install.sh)"

# brew
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

# vscode可能需要镜像
http://vscode.cdn.azure.cn/stable/3866c3553be8b268c8a7f8c0482c0c0177aa8bfa/VSCode-darwin-universal.zip

# brew可以并行安装以下所有内容
brew install  # 后面跟下面的每一个
v2ray
google-chrome		
karabiner-elements	
omnidisksweeper 
qlcolorcode	
qlcommonmark
qlimagesize	
qlstephen
quicklook-csv		
quicklook-json
suspicious-package
textmate
typora	
upic
visual-studio-code

# 启动v2ray
v2ray -config=/usr/local/etc/v2ray/config.json # 先执行这个, 看看是否有问题
brew services start v2ray # 然后搞成service执行

```
